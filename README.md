# Романтичное приглашение для Анастасии

Одностраничный сайт-приглашение на свидание в розовых тонах с анимациями.

## Как изменить сюрприз

Откройте файл `script.js` и измените объект `SURPRISE` в начале файла:

```javascript
const SURPRISE = {
  place: 'Кафе «Роза» на Невском, 42',
  datetime: 'Суббота, 5 июля, в 19:00',
  note: 'Розовое платье — необязательно, но будет мило 💕',
};
```

После изменения сохраните файл и обновите сайт (см. ниже).

## Как обновить сайт после правок

```bash
git add .
git commit -m "Обновить детали свидания"
git push
```

Изменения появятся на GitHub Pages через 1–2 минуты.

## Локальный просмотр

Откройте `index.html` в браузере или запустите простой сервер:

```bash
npx serve .
```

## Публикация на GitHub Pages

1. Создайте публичный репозиторий на GitHub
2. Загрузите файлы: `git push -u origin main`
3. В репозитории: **Settings → Pages → Source: Deploy from branch → `main` / `/ (root)`**
4. Сайт будет доступен по адресу: `https://<ваш-username>.github.io/<имя-репозитория>/`

## Структура

- `index.html` — разметка страницы
- `styles.css` — стили и анимации
- `script.js` — интерактив и данные сюрприза
